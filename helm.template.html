Operator v1

Charts to install LINBIT SDS for Kubernetes.

These charts pull LINBIT customer only container images.

Usage

Detailed instructions can be found in the LINSTOR User Guide.


$ helm repo add linstor https://charts.linstor.io
$ helm install linstor-op linstor/linstor --set etcd.enabled=false --set operator.controller.dbConnectionURL=k8s
